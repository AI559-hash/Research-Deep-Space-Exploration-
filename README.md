# Research-Deep-Space-Exploration-
# Galaxy Morphological Classification using ML & Deep Learning

  ## Overview
  This research project focuses on classifying galaxy morphologies based on
  astronomical images using Machine Learning and Deep Learning techniques. Conducted
  under the supervision of **Dr. Muhammad Umair (PhD)** at the **University of
  Central Punjab, Lahore**.

  Galaxy morphological classification is a critical task in astrophysics, helping
  astronomers understand galaxy formation, evolution, and structure. This project
  aims to automate this classification process using image-based ML/DL models.

  ## Data Preprocessing Pipeline
  The preprocessing phase involved preparing raw galaxy image data for model
  training:

  - **Image Resizing** — Standardized all galaxy images to uniform dimensions for
  consistent model input.
  - **Normalization** — Scaled pixel values to a standard range to improve model
  convergence.
  - **Noise Reduction** — Applied filtering techniques to remove artifacts and
  improve image clarity.
  - **Data Augmentation** — Used rotation, flipping, and brightness adjustments to
  increase dataset diversity and reduce overfitting.
  - **Label Encoding** — Mapped galaxy morphology categories to numerical labels for
  supervised learning.
  - **Train-Test Splitting** — Divided the dataset into training, validation, and
  test sets for robust evaluation.
